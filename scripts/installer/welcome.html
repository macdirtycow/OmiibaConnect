Omiiba Connect

Sony Headphones macOS Client — unofficial companion for WH-1000XM3 / XM4 / XM5 / XM6.

This package installs or updates Omiiba Connect in your Applications folder.

Already installed? You do not need to move the old app to the Trash. Quit Omiiba Connect (or let the installer quit it), then continue — the new version replaces the old one automatically.
